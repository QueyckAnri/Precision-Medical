Design a premium desktop 'Second Opinion' page for the neurosurgery clinic 'Precision Medical'.

**DESIGN SYSTEM (REQUIRED):**
- Strict Swiss grid layout, dark charcoal text #262323 on warm light gray background #F9F9F9
- Bold display headings using monospace styling, all uppercase, tight line height
- Borders should be extremely thin, using rgba(38,35,35,0.08) hairline dividers
- Call to action buttons should be perfectly rounded pill-shapes in deep orange #F55F24
- Data sections and stats grids should look like technical catalogs or grids, clean and structured
- NO rounded corners on containers — only on pill-buttons and circular avatars
- Accent color #F55F24 used only for CTAs, hover states, and key icons

**Page Structure:**
1. Shared site header (logo + nav: Departments, About Us, Our Surgeons, Second Opinion, Contacts)
2. Hero: 'SECOND OPINION' large Tiposka uppercase display heading + subtitle about remote expert consultation
3. Upload Zone: Large drag-and-drop area with dashed border for MRI/CT scan files (DICOM, JPEG, PDF accepted)
4. Case Description Form: Patient name, diagnosis, current treatment, hospital, question for specialists. Bottom-border only inputs per design system.
5. Specialist Selection: List of available surgeons to direct the second opinion to (use same 5 surgeons as Our Surgeons page)
6. Submission confirmation state and privacy notice
7. Bottom CTA: 'Submit Case Files' primary pill button in #F55F24
