# Site Vision: Precision Medical

High-precision neurological surgeries and neurosurgery consultation. Modern, premium, clean Swiss/minimalist design system with high-contrast text layout, custom monospace headings, and interactive 3D elements.

## 1. Project Information
- **Stitch Project ID:** `12282722675370129022`
- **Platform:** Vite static HTML / CSS / Vanilla JS project

## 2. Design Goals
- High-contrast Neo-Grotesque styling (Helvetica Neue body, Tiposka display font).
- Swiss/minimalist layout structure with bold borders (`rgba(38, 35, 35, 0.08)`), generous padding, and flat cards (`rgba(38, 35, 35, 0.04)`).
- Distinctive primary brand color `#F55F24` (Deep Orange/Coral) for accents, links, and CTAs.
- Seamless page transitions (slide-up / slide-down) using DOM class triggers and CSS animations.
