How to install custom fonts:

To display the brand typography correctly, place the font files in this folder with the following names:

1. Tiposka (for headings):
   - Tiposka.woff2
   - Tiposka.woff
   - Tiposka.ttf

2. HelveticaNeueCyr (for main text):
   - HelveticaNeueCyr-Roman.woff2
   - HelveticaNeueCyr-Roman.woff
   - HelveticaNeueCyr-Roman.ttf

The CSS styles in style.css are already configured to look for these files. If they are not found, the website will fall back to system fonts.
